## Setup

```bash
npm install
make dev
```

## Commands

```bash
make dev          # Start dev server
make build        # Production build
make generate     # Static site generation
make lint         # ESLint + typecheck
make lint-fix     # Auto-fix lint issues
make test         # Run tests
```
